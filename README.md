# 🔍 Question Answer Search

A Question Answer Search application built using **Scikit-learn**, **FastAPI**, and **Streamlit**. The project uses a dataset of AI, Machine Learning, and Data Science question-answer pairs from Kaggle. User questions are converted into TF-IDF vectors, and a **Nearest Neighbors** model finds the most similar question in the dataset. The corresponding answer is then returned.

**Files:**

* `test1.ipynb` – Data preprocessing, TF-IDF vectorization, model training, and saving artifacts.
* `api.py` – FastAPI backend for prediction.
* `app.py` – Streamlit frontend for user interaction.

Dataset: https://www.kaggle.com/datasets/nvnikhil0001/data-science-questions-and-answers
